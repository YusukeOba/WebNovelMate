## Getting started

1. Install Flutter. See <https://flutter.io/setup/>
2. Setting up the IntelliJ. See <https://flutter.io/ide-setup/>
3. Fork and clone this repository.
4. Move to ``WebNovelMate`` directory.
5. Run `flutter run` command.
